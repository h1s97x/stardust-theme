# Stardust Theme

> A VS Code theme inspired by the cosmos ✨

![Version](https://img.shields.io/badge/version-0.2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![VS Code](https://img.shields.io/badge/VS%20Code-1.74%2B-blue)

## 🌟 Features

- **Deep dark background** inspired by the night sky
- **Light variant** for daytime coding
- **Carefully selected colors** for long coding sessions
- **Optimized readability** with reduced eye strain
- **Multi-language support** with syntax highlighting

## 🎨 Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Background (Dark) | Deep Space | `#0d1117` |
| Foreground (Dark) | Starlight | `#c9d1d9` |
| Keywords | Mars Red | `#ff7b72` |
| Strings | Sky Blue | `#a5d6ff` |
| Functions | Violet | `#d2a8ff` |
| Numbers | Ice Blue | `#79c0ff` |
| Classes | Amber | `#ffa657` |
| Comments | Nebula Gray | `#8b949e` |

## 📦 Installation

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for **"Stardust Theme"**
4. Click **Install**
5. Select the theme: `Preferences > Color Theme > Stardust`

## 🖼️ Themes Included

- **Stardust** - Dark theme for night coding
- **Stardust Light** - Light theme for day coding
