# Stardust Theme

> A dark VS Code theme inspired by the cosmos

![Theme Preview](https://raw.githubusercontent.com/h1s97x/stardust-theme/main/assets/preview.png)

## Features

- Deep dark background inspired by the night sky
- Carefully selected color palette for long coding sessions
- Optimized for readability and reduced eye strain
- Support for multiple programming languages

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Stardust Theme"
4. Click Install
5. Select the theme: `Preferences > Color Theme > Stardust`

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#0d1117` | Editor background |
| Foreground | `#c9d1d9` | Main text |
| Keyword | `#ff7b72` | Keywords, operators |
| String | `#a5d6ff` | Strings |
| Function | `#d2a8ff` | Functions |
| Number | `#79c0ff` | Numbers, constants |
| Class | `#ffa657` | Classes, types |
| Comment | `#8b949e` | Comments |
